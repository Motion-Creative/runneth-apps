Health Alerts

Monitors all connected tools and active routines. Sends Slack alerts when integrations break, routines go overdue, dependency chains fail, or skill execution cannot be confirmed.

What this enables

🔌

Connected tool monitoring

5-layer checks on every connected tool — Motion, GitHub, Slack, Notion, Linear, Dropbox, OpenAI, LaunchDarkly and more. Plain-language alerts when something stops working.

⚡

Routine health checks

Every scheduled task checked every 30 minutes. Detects overdue routines, silent drops from the reminder system, broken dependency chains, and incomplete executions.

🔔

Smart alerts — no noise

One alert when something breaks. One follow-up after your configured window if still unresolved. A confirmation when things recover. Nothing else.

🔍

Missed routine detection (opt-in)

Daily scan of conversations catches tasks requested but never created. Self-resolves when parameters are clear, asks for details when they are not.
