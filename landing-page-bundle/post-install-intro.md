# Landing page bundle

## What just opened up
You now have the full landing-page stack: a recreation-quality page summary skill, an extractable brand kit, a CRO audit with top fixes, and a scored A/B test backlog. All four work together. The summary is the upstream evidence layer; brand-kit, optimize, and experiments each call it fresh on every run, so nothing downstream goes stale.

## Try this now
1. **Run the full stack on your main page**: `Summarize [landing-page-url], build a brand kit from it, audit it for CRO, and give me an experiment backlog.`
   _You'll get back:_ summary written to the brain, brand kit produced as markdown + HTML preview, CRO audit with top 3 fixes, and a scored experiment backlog. Four files saved, ready to act on.
2. **Just the brand kit**: `Build a brand kit for [brand-url].`
   _You'll get back:_ a complete kit Runneth will auto-consult on every future branded output (decks, briefs, ads, HTML pages).
3. **Just the CRO audit**: `Audit [landing-page-url] for CRO against this ad copy: [paste].`
   _You'll get back:_ message-match analysis, top 3 fixes by lift x ease, and a short test queue.
