# Usage efficiency audit

## What just opened up
The team now has a quiet watcher on Runneth usage. Every day Runneth checks where the team is in the cycle. The moment usage crosses 80%, a Slack note lands in the configured channel with a friendly read on a few habit shifts that would unlock more out of each conversation. The voice throughout is a teammate helping the team get more value, never an alert about a limit. It only fires once per cycle, so the team is never nagged.

## Try this now
1. **Force a preview right now**: `Run the usage audit on what we've used so far this cycle.`
   _The team gets back:_ a personalized page based on real conversations from the current cycle. Treat this as a sneak peek of what would land at 80% later.
2. **Change where the heads-up posts**: `Reconfigure the usage audit.`
   _The team gets back:_ a chance to update the Slack channel, who gets tagged, or the billing anniversary day.
3. **Check where the team is right now**: `How are we tracking on usage this cycle?`
   _The team gets back:_ percentage of plan used and days until reset, no audit attached.

## Compounds with
- **bootcamp:** New teammates getting onboarded learn good Runneth habits early, so the team rarely needs the audit in the first place.
