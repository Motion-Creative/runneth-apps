# Format By Purpose

Format is the production structure of the ad. The right format depends on what the message needs to do, which depends on the awareness stage. Format choice is not arbitrary.

This file bundles the format-purpose-stage mapping from Alysha's framework. The `visual-formats` skill (also bundled in this pack) catalogs 45+ specific formats with definitions. This file is the strategic bucketing on top of that catalog — which categories of format fit which awareness stages, and why.

## The principle

Match the format's purpose to the awareness stage's goal. If the stage calls for revealing a problem, use formats that reveal. If the stage calls for proving superiority, use formats that prove.

Any format can technically work at any stage when the messaging is right. But the categories below are the high-percentage matches.

## The four format-purpose categories

### Formats that educate and reveal

**Best for:** Unaware, Problem-Aware

These formats interrupt, observe, and explain. They earn attention by making the viewer notice something they hadn't noticed about themselves or the problem.

Examples from the `visual-formats` skill:

- Storytelling
- Before/After
- Brain Scan View
- ASMR
- Pattern Interrupt (broadly)

The job: surface a problem the viewer hasn't acknowledged, or reframe a familiar problem so they notice the cost.

### Formats that compare and demonstrate

**Best for:** Solution-Aware, Product-Aware

These formats put options side by side or show the product in action. They earn attention by helping the viewer evaluate.

Examples:

- Talking Head
- Greenscreen Library
- Vs./Them (Us Vs Them)
- How To
- Feature/Benefit Portrait

The job: position this solution as credible, distinct, and obviously the right pick among alternatives.

### Formats that prove and build trust

**Best for:** Problem-Aware through Most-Aware

These formats lean on third-party validation, real users, or accumulated social proof. They earn attention through credibility — someone like the viewer is showing them this works.

Examples:

- Testimonial / UGC
- Social Proof Montage
- Consumer Response

The job: reduce risk by demonstrating that others like the viewer have already crossed the line and benefited.

### Formats that drive action

**Best for:** Product-Aware, Most-Aware

These formats are direct, transactional, and structured for conversion. They earn attention through clarity of the offer.

Examples:

- Static (offer-first)
- Text Message format
- Direct Response (offer-led)

The job: close the loop. Remove final friction and trigger the click.

## How to use this with concepting and briefing

When building a concept (via `creative-strategy-engine` and `creative-mechanics`) or a brief (via the `briefing` skill in `/runneth/skills/`):

1. **Identify the awareness stage** from the strategy file.
2. **Pick the format-purpose category** that matches that stage's goal.
3. **Browse `visual-formats` for specific formats** inside that category, considering medium constraints (video vs static), brand resources, and mechanic compatibility.
4. **Verify the pairing.** A Pain Agitation lens at Problem-Aware fits an "educate and reveal" format. An Objections lens at Product-Aware fits a "compare and demonstrate" or "prove and build trust" format.

## Bidirectional choice

Format and mechanic are bidirectional. You can start with a format and find the right mechanic, or start with a mechanic and find the format that best delivers it. Both workflows are valid (this is documented in the `visual-formats` skill and the `creative-mechanics` skill). Use this file to keep the format-stage match honest in either direction.

## Anti-pattern

Picking formats by what's trending without checking the stage fit. A category-trending UGC format used at Unaware will fail because the viewer doesn't yet know they need to listen to a testimonial. Reverse: a Demo format at Solution-Aware will sometimes work and sometimes feel premature depending on how problem-acknowledged the segment is. Always cross-check stage and format category.
