# Hook Evaluator Skill

This skill evaluates the hook (first 3–5 seconds) of any video ad from structured text inputs — no video file required. It uses the Creative Strategy Engine framework and hook psychology taxonomy to produce a full diagnostic of why a hook works (or doesn't), what tactic and trigger it deploys, what awareness stage it's targeting, and how it would perform in feed.

---

## Required Inputs

Before evaluating, you need four inputs from the user. If any are missing, ask for them before proceeding.

### 1. Visual Description
A description of what's happening on screen in the first 3–5 seconds. Should include:
- The scene or setting
- Who or what is in the frame
- Any notable action, movement, or visual moment
- Camera angle or framing if notable (close-up, POV, overhead, etc.)
- Any visual contrast, color, or production style details

### 2. On-Screen Text
The exact words of any text overlaid on the video in the hook window. Should include:
- The exact copy (word for word if possible)
- Placement on screen (top, center, bottom, over a person's face, etc.)
- Any styling notes that affect meaning (all caps, large font, stacked lines, etc.)

If there is no on-screen text, the user should note that explicitly.

### 3. Voiceover / Spoken Copy
The exact words spoken in the hook — whether by someone on camera, a voiceover, or a text-to-speech voice. Should be a verbatim transcript if possible.

### 4. Voice / Audio Treatment Description
A description of how the audio sounds and feels, including:
- Type of voice (e.g., female, male, young, authoritative, casual, conversational, whispery)
- Tone and delivery (e.g., urgent, calm, confessional, excited, deadpan, direct, warm)
- Any notable audio treatment (e.g., echo, lo-fi, background music, silence, sound effect)
- Pacing (fast, slow, deliberate)
- Whether the voice sounds like a creator, a narrator, a real person, or a polished voiceover

## Evaluation Output

Once you have all four inputs, produce the full evaluation in the following seven-part structure.

---

### PART 1: HOOK RECONSTRUCTION

Before analyzing, synthesize what the viewer actually experiences. Describe the first 3–5 seconds as a unified experience — what they see, hear, and read simultaneously, and how those channels work together or against each other.

Answer:
- What is the viewer's attention being pulled toward first — the audio, the visual, or the text?
- Are the three channels saying the same thing, layering meaning, or creating contrast?
- Is there any friction or disconnection between channels that might create intrigue or confusion?

---

### PART 2: HOOK TACTIC IDENTIFICATION

Identify the primary hook tactic being used. Choose from:

**Aspirational, Authority, Belief, Bold Claim, Call To Action First, Challenge, Confession, Contrast, Contrarian, Curiosity, Demographic Callout, Direct Address, Directive, Exclusivity, Explainer, FOMO, How To, If Then, Listicle, Myth Busting, Offer Only, Price Anchor, Question, Reasons Why, Relatability, Reverse Psychology, Risk Reversal, Shocking Statement, Social Proof, Statistic, Storytelling, Urgency, Warning**

For each tactic identified:
- Name it
- Define it in one sentence
- Explain why this tactic was deployed for this audience and product context
- Note any secondary tactic that's layered in

---

### PART 3: PSYCHOLOGICAL TRIGGER ANALYSIS

Identify which psychological trigger(s) are being activated. Choose from:

| Trigger | What It Does |
|---|---|
| **Pattern Interrupt** | Breaks autopilot scrolling — something unexpected, counterintuitive, or visually jarring |
| **Identity Call-Out** | Makes a specific viewer immediately self-select — feels like it was made for them |
| **Pain Agitation** | Mirrors the viewer's internal experience back at them — makes them feel deeply understood |
| **Curiosity Gap** | Creates an open loop they need to close — a question they can't stop thinking about |
| **Social Proof / Credibility** | Builds trust through results, volume, or expertise |
| **Contrarian / Myth-Busting** | Challenges a belief they hold — makes them feel like they've been lied to |
| **Aspiration / Desire** | Shows them the version of themselves on the other side of the product |
| **Urgency / Stakes** | Makes inaction feel costly |

For each trigger:
1. Name it
2. Point to the specific element (a word, visual beat, audio moment, or text phrase) that activates it
3. Explain why this trigger works for this audience — what emotional response is it designed to produce?

---

### PART 4: AWARENESS STAGE DIAGNOSIS

Identify which of the five awareness stages this hook is targeting:

| Stage | Who They Are | What the Hook Should Do |
|---|---|---|
| **Unaware** | Doesn't know they have a problem | Introduce pain/desire through relatable situation. No product mention. |
| **Problem-Aware** | Knows the problem, hasn't found a solution | Agitate the pain hard. Make them feel deeply understood. |
| **Solution-Aware** | Knows solutions exist, comparing options | Differentiate. Position against alternatives. Call out what's failed them. |
| **Product-Aware** | Knows the product, hasn't bought | Remove objections. Trigger FOMO. Counter the reason they haven't bought. |
| **Most-Aware** | Ready to buy, needs a nudge | Direct offer, urgency, guarantee, price anchoring. CTA-forward. |

Point to specific evidence from the inputs — language, framing, level of product mention, presence or absence of a solution — that signals the awareness stage. If the hook straddles two stages, name which is primary and why.

---

### PART 5: FEED NATIVE ANALYSIS

Evaluate how this hook would perform in a real social media feed across five dimensions:

**Scroll stop probability**
Does this feel like organic content or does it immediately read as an ad? What is the specific element most likely to make a thumb stop?

**Native feel**
Does the visual style, pacing, audio treatment, or format blend into the feed in a way that earns attention before the viewer realizes it's an ad? Or does it signal "ad" from frame one?

**First-frame power**
If someone saw only the first frame with no audio — would they stop? What's carrying the hook visually?

**Sound-off performance**
If someone is scrolling with their phone on silent, does the hook still work? Is the on-screen text and visual enough to stop them and pull them in?

**Channel fit**
Based on the visual style, audio treatment, and pacing — which platform does this feel native to? (TikTok, Instagram Reels, Facebook Feed, YouTube pre-roll, etc.) Would it need to be adapted for other placements?

---

### PART 6: MESSAGING ANGLE REVERSE-ENGINEERING

Based on all four inputs, identify the likely messaging angle this hook was built on. A messaging angle is the core truth at the intersection of a specific pain (or desire) and a specific persona.

Answer each:

**Pain or Desire Anchor**
What problem is this hook built around? Or what desire is it activating? Is it pain-first or desire-first?

**Persona**
Who is the specific person this hook was written for? Don't just name a demographic — describe their life context. What are they doing, feeling, or struggling with that makes this hook land for them specifically?

**Messaging Angle**
Articulate the core truth in one sentence. This is the emotional center of gravity — the thing that, if you stripped away every tactic and trigger, is still true about why this product matters to this person.

*Examples of well-formed messaging angles:*
- "You shouldn't have to cancel plans because of your skin."
- "You're spending $50K/month on ads and flying completely blind."
- "Your dermatologist wrecked your skin and called it treatment."

---

### PART 7: OVERALL HOOK EFFECTIVENESS SCORE

Score this hook out of 10 using the rubric below. Show the score breakdown, then the total.

| Dimension | Max Points | Score | Notes |
|---|---|---|---|
| **Clarity** — Is the viewer immediately clear on who this is for and what it's about? | 2 | | |
| **Emotional resonance** — Does it produce a felt response? Does it feel like it was made for the viewer? | 2 | | |
| **Feed nativity** — Does it earn attention before it reads as an ad? | 2 | | |
| **Strategic accuracy** — Is the tactic and trigger right for this awareness stage and persona? | 2 | | |
| **Execution quality** — Do the spoken, visual, and text channels work together? Any element that undercuts the hook? | 2 | | |
| **TOTAL** | **10** | | |

End with:
- **What this hook does best** — one sentence
- **Highest-leverage improvement** — one specific, actionable change that would make this hook stronger

## Notes on Input Quality

The quality of your evaluation is only as good as the inputs. If a description is vague or missing detail, flag it before completing the evaluation. For example:

- If the visual description doesn't say what's in frame, ask what the viewer actually sees
- If the voice description only says "female voice," push for tone and delivery style
- If on-screen text placement isn't noted, ask where it sits — placement affects hierarchy and read order

You can produce a partial evaluation with what's available and note where gaps are limiting the diagnosis, but flag it clearly.

## Integration Notes

This skill operates downstream of the **Creative Strategy Engine**. If the user has brand context available (persona, pain anchor, messaging angle, awareness stage target), note it at the top of your evaluation and use it to sharpen Parts 4 and 6. Without it, reverse-engineer from the hook itself.

This skill complements **Hook Writing** — if the evaluation reveals the hook is underperforming, use the Hook Writing skill to produce improved alternatives using the same messaging angle and awareness stage.
