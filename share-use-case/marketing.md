## Super powers this unlocks

- Turns "I built this thing" into a clean handoff in a minute.
- Bundles the files, the install steps, and a PR description.
- Writes the description in plain English, not internal jargon.
- Lets a teammate install in a fresh sandbox without asking you a single question.

## How it works

Run the skill in the sandbox where the use case lives. Your Runneth scans what got built, packages the files into a zip, and writes a step-by-step submission guide plus a pre-filled PR description, ready to drop into the shared library.

## A real example

Sophia builds a tight CSM digest for her own book. Two weeks later three other CSMs want the same thing. She runs share-use-case once. Her Runneth produces a zip, a one-page how-to, and a draft PR description. She uploads it to the library. The other CSMs install it the same afternoon, no Slack thread, no questions for Sophia.
