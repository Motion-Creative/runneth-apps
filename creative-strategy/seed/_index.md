# Creative Strategy bundles

Local routing index for per-brand creative-strategy context produced by this package.

Each brand gets one bundle at `/agent/brain/creative-strategy/<workspace-slug>/`:

- `brand-context.md` (brand-intake)
- `product-catalog.md` (product-catalog)
- `review-audit.md` (review-audit)
- `keywords.md` (brand-relevant-keywords)
- `competitor-analysis.md` (competitor-analysis)
- `messaging-angles.md` (creative-strategy-engine)

Read the relevant file before creative work for that brand. Rebuild a file only when the brand, products, reviews, or competitors actually change.
