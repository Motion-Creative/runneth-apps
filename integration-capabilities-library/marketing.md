## Super powers this unlocks

- Answers "can we do X with Y?" with a real source.
- Names the exact permission scope each call needs.
- Catches API changes the next morning, not the next quarter.
- Keeps the team from rediscovering the same limits twice.

## How it works

Every night your Runneth refreshes a structured reference for every connected integration: what each one can do, what scopes it needs, and where the docs say so. Future answers cite the line in the reference, with provenance.

## A real example

Reza is scoping a new automation that depends on HubSpot deals. He asks his Runneth whether the current connection can move a deal between stages. The answer comes back yes, with the specific scope used, the API call name, and a link to the section in the reference updated last night. No detour into the HubSpot docs.
