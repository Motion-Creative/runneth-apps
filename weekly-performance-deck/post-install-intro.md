# Weekly performance deck

## What just opened up
Every Friday, Runneth produces a brand-styled HTML deck published as a dated app route, framed against last week's snapshot so the narrative compounds. It composes the paid-strategy briefs for funnel framing, creative-deep-dive for per-creative diagnoses, and brand-kit for visual style. Routes accumulate over time so you can scroll back through the full performance story week by week.

## Try this now
1. **Generate this week's deck right now**: `Generate the weekly performance deck for this week.`
   _You'll get back:_ the deck builds, publishes at `/agent/apps/weekly-performance-deck-<workspace-slug>/routes/<date>/`, and you get an openable app link to share with the team.
2. **Re-run for last week**: `Generate the weekly deck for week of [date].`
   _You'll get back:_ a dated route for that week, framed against the week before it, narrative continuity preserved.
3. **Confirm the Friday reminder is scheduled**: `Is the weekly deck reminder set up?`
   _You'll get back:_ confirmation of the Friday 2pm reminder, the workspace slug it runs against, and what to do if you need to change cadence.

## Compounds with
- **brand-audit:** Brand strategy feeds the narrative frame.
- **paid-strategy-audit:** Funnel framing per channel.
- **creative-deep-dive:** Per-creative diagnoses on standout ads.
- **brand-kit:** Visual style and voice.
